FEATURE 3 — Analysis sharing via link

--- DATABASE ---

New table in admin.db:

TABLE shared_analyses:
  id            INTEGER PRIMARY KEY AUTOINCREMENT
  token         TEXT UNIQUE NOT NULL  
    (UUID v4, generated on share creation)
  analysis_id   INTEGER REFERENCES analysis_log(id)
  created_by    INTEGER REFERENCES users(id)
  created_at    DATETIME DEFAULT CURRENT_TIMESTAMP
  expires_at    DATETIME NOT NULL
  is_active     BOOLEAN DEFAULT 1
  title         TEXT     
    (optional custom title for the share)
  view_count    INTEGER DEFAULT 0

TABLE share_access_log:
  id            INTEGER PRIMARY KEY AUTOINCREMENT
  share_id      INTEGER NOT NULL 
    REFERENCES shared_analyses(id)
  accessed_at   DATETIME DEFAULT CURRENT_TIMESTAMP
  ip_hash       TEXT    
    (SHA256 of IP, anonymized per DSGVO)
  user_agent    TEXT

New settings entry:
  sharing_base_url = ''  
    (e.g. 'https://gaio-analyzer.replit.app'
    or 'https://analyse.meinefirma.de')
  sharing_default_expiry_days = '30'
  sharing_enabled = 'true'

--- BACKEND ---

  POST /api/admin/shares
    Admin only.
    Body: { 
      analysisId: number,
      expiryDays?: number,  (default from settings)
      title?: string 
    }
    Generates UUID token.
    Inserts into shared_analyses.
    Returns { token, shareUrl, expiresAt }
    
    shareUrl = [sharing_base_url]/share/[token]
    If sharing_base_url is empty, use 
    the request's own origin as base.

  GET /api/shares/:token  (PUBLIC, no auth)
    Checks token exists, is_active=true, 
    not expired.
    Logs access to share_access_log 
    (hashed IP, user agent).
    Increments view_count.
    Returns the full analysis data needed 
    to render results (same structure as 
    the existing analysis result object).
    Returns 404 if not found.
    Returns 410 Gone if expired.

  GET /api/admin/shares
    Admin only.
    Returns all shares with stats:
    token, title, domain, createdAt, 
    expiresAt, isActive, viewCount,
    lastAccessed (from share_access_log)

  DELETE /api/admin/shares/:id
    Admin only. Sets is_active=false.

  GET /api/admin/shares/:id/access-log
    Admin only.
    Returns paginated access log for 
    one share: accessed_at, ip_hash 
    (show as "Besucher [hash[:6]]"), 
    user_agent (parse to browser/OS name)
